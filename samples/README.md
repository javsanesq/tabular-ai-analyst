# Demo Datasets

This repository includes small demo subsets for repeatable local tests and screenshots.

- `owid_co2_subset.csv` is adapted from Our World in Data CO2 and Greenhouse Gas Emissions materials. Cite OWID and the underlying Global Carbon Project source when using the full dataset.
- `video_games_subset.csv` is a tiny synthetic portfolio demo inspired by common video-game sales schemas. It is curated for semantic ranking and filter tests, not copied from a full third-party dataset.
- `wine_quality_subset.csv` is adapted from the UCI Wine Quality dataset by Cortez et al. The UCI dataset is licensed under CC BY 4.0.

These files are intentionally small and are not meant to replace the full source datasets.
